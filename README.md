# ePortfolio Week 7

This project extends my ePortfolio by connecting a React frontend to a Node.js/Express backend using MongoDB and Mongoose. The Projects section now dynamically loads data from a database instead of using static JSON.

---

## Features

- MongoDB local database connection using Mongoose
- Environment variables for secure configuration
- Project schema and model with validation
- Full CRUD API for portfolio projects
- React frontend dynamically fetches project data
- Separation of concerns using models, controllers, and routes
